# 钢材进销存管理系统 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个基于Flask+MySQL8的钢材进销存管理系统，以卡号为唯一追踪标识，覆盖采购入库、加工管理、销售出库、库存管理和完整报表体系。

**Architecture:** Flask应用工厂模式，Blueprint模块化，服务层封装业务逻辑，Jinja2+HTMX+TailwindCSS服务端渲染，MySQL8存储。

**Tech Stack:** Python 3.10+, Flask, Flask-SQLAlchemy, Flask-Login, PyMySQL, HTMX, TailwindCSS (CDN), Chart.js (CDN), openpyxl
