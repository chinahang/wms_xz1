业务流程设计

采购入库 → 加工管理 → 销售出库 的核心操作路径

📥 采购入库流程

1.新建入库单
→
填写供应商、销售部门、主单备注
→
2.录入明细
→
逐行录入卡号、品名、牌号、产地、规格、件数、吨位、备注
→
3.保存
→
自动汇总总件数/总吨位 → 同步创建库存记录 → 写入流水

⚙️ 加工管理流程（一对多）

1.新建加工单
→
2.选原料卡号
→
3.录入产成品
→
4.保存
一对多加工示例：
原料卡号
KH-2026001
品名: 热轧卷板
规格: 6.0×1500
吨位: 28.5吨
产成品① 新卡号
KH-2026002
品名/牌号/产地/规格 手工录入
吨位: 15.2吨
产成品② 新卡号
KH-2026003
品名/牌号/产地/规格 手工录入
吨位: 11.8吨
损耗
手工录入
1.5吨
保存后：原料卡号标记已消耗 → 产成品卡号创建库存 → 写入流水

📤 销售出库流程

1.新建出库单
→
填写客户、销售部门
→
2.选卡号出库
→
从库存中搜索/选择卡号，自动带入品名/规格等信息，可修改出库吨位
→
3.保存
→
汇总总件数/总吨位 → 库存标记已出库 → 写入流水

出库明细报表
按日期/客户/品名/部门筛选，支持导出Excel
